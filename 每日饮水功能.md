"每日饮水记录"功能已开发完成，所有服务正常运行。以下是改动摘要：

  ---
  数据库

  新建表 water_logs（database/migration_add_water_logs.sql）
  ┌─────────────┬──────────┬──────────────────────┐
  │    字段     │   类型   │         说明         │
  ├─────────────┼──────────┼──────────────────────┤
  │ id          │ INT PK   │ 自增主键             │
  ├─────────────┼──────────┼──────────────────────┤
  │ user_id     │ INT FK   │ 关联 users 表        │
  ├─────────────┼──────────┼──────────────────────┤
  │ amount_ml   │ INT      │ 饮水量（默认 250ml） │
  ├─────────────┼──────────┼──────────────────────┤
  │ record_date │ DATE     │ 记录日期             │
  ├─────────────┼──────────┼──────────────────────┤
  │ created_at  │ DATETIME │ 创建时间             │
  └─────────────┴──────────┴──────────────────────┘

  ---
  后端（backend/routes/water.js）

  ┌──────────────────────┬─────────────────────────────────────────┐
  │         API          │                  说明                   │
  ├──────────────────────┼─────────────────────────────────────────┤
  │ POST /api/water/add  │ 增加饮水量（默认 +250ml），返回今日总量 │
  ├──────────────────────┼─────────────────────────────────────────┤
  │ GET /api/water/today │ 获取今日总饮水量                        │
  └──────────────────────┴─────────────────────────────────────────┘

  需要 JWT 认证，已注册到 app.js。

  ---
  前端（frontend/index.html + style.css）

  在首页仪表盘 stats-grid 下方新增了 "今日饮水目标"卡片：
  - 8 个小水杯 排成一行，每个代表 250ml，目标 2000ml
  - 点击 "+250ml" 按钮调用接口记录饮水
  - 已达标的水杯变为 蓝色填充 + 水滴图标 💧
  - 右侧实时显示 750 / 2000 ml 数字进度
