学习路线（按依赖关系排序）

第一步：HTML + CSS + JavaScript（前端的骨架）

这是所有网页的基础，这个项目的用户前端就是用它们手写的。

- 怎么学：打开 MDN 教程，一边看一边改你项目里的 index.html / style.css /
  main.js
- 入门资源：MDN Web 入门教程 (https://developer.mozilla.org/zh-CN/docs/Learn/
  Getting_started_with_the_web)（免费、中文、权威）
- 在这个项目里练手：试着改一下首页公告横幅的文字颜色，或者给按钮换个样式

第二步：Node.js + Express.js（后端）

Node.js 让你用 JavaScript 写服务器，Express.js 是它最流行的框架。你的 app.js
就是入口。

- 怎么学：理解"请求 → 路由 → 控制器 → 数据库"这个链条
- 入门资源：Express 官方中文指南
  (https://expressjs.com/zh-cn/guide/routing.html)
- 在这个项目里练手：打开 backend/app.js，看第 27 行 app.get('/api/health',
  ...) 是怎么响应一个健康检查请求的；试着加一个 /api/ping 路由返回 pong

第三步：MySQL + SQL 基础

你的所有数据（用户、食物、饮食记录）都存在 MySQL 数据库里。

- 需要知道的：表是什么、SELECT/INSERT/UPDATE/DELETE 怎么写、WHERE
  条件怎么过滤
- 入门资源：SQL 教程 - 菜鸟教程
  (https://www.runoob.com/sql/sql-tutorial.html)（简洁，够用）
- 在这个项目里练手：打开 database/ 目录下的 SQL 文件，看看建表语句；连上本地
  MySQL 跑一句 SELECT \* FROM food_items LIMIT 10 感受一下

第四步：JWT 认证

JWT 就是一串加密的字符串，用户登录后服务端发给他，之后每次请求带上这个串就能
证明"我是我"。

- 类比理解：就像游乐园的手环，进门时给你戴上，之后玩项目出示手环即可，不用每
  次都查身份证
- 入门资源：JWT 图文解释 (https://jwt.io/introduction)（有交互式调试器）
- 在这个项目里练手：打开 backend/middleware/auth.js，看它是怎么从请求头里取出
  Token 并验证的

第五步：Vue 3 + Element Plus（后台管理）

你的 admin-frontend/ 就是用这套搭的。Vue 负责页面逻辑，Element Plus
提供现成的按钮、表格、弹窗等组件。

- 怎么学：先跑起来 cd admin-frontend && npx vite，打开浏览器看效果
- 入门资源：Vue 3 官方教程
  (https://cn.vuejs.org/tutorial/#step-1)（交互式，边看边写）
- 在这个项目里练手：打开
  admin-frontend/src/views/Announcement.vue，看看一个管理页面是怎么用表格 +
  表单 + 弹窗组成的

第六步：AI 接口调用

你的项目调用了 DeepSeek 和 Qwen-VL-Max 两个大模型。本质上就是发一个 HTTP
请求给 AI 服务商，把用户问题送过去，把 AI 回复取回来。

- 在这个项目里练手：打开 backend/controllers/aiController.js，重点看 callAI()
  函数（第 277-350 行），理解"拼消息 → fetch 发请求 → 解析返回内容"三步

一个具体的小任务帮你串起来

试着做一个"留言板"小功能，横跨整个链路：

1. 数据库加一张 messages 表
2. 后端写一个 POST /api/messages 接口
3. 用户前端写一个表单提交留言
4. 后台管理用 Vue 写一个留言列表页

这个做完，你就能把上面的技术全部摸一遍了。你的项目已经搭好了骨架，直接在上面
加就行，会比从零开始快很多。
