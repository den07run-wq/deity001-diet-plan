项目现状总结
  已完成

  项目核心闭环已完整打通：用户注册登录 → 录入身体数据 →
  创建饮食计划 → 每日记录饮食 → 营养分析报告，外加 AI
  智能录入（文本 + 图片识别）和
  管理后台（用户管理、食材管理、公告管理、操作日志）。约 40+
  个 API 端点，200+ 种预置食材。

  最明显的缺失

  没有个人中心/设置页面。PUT /api/user/profile 接口已就绪，但
  前端没有任何地方让用户修改性别、年龄、身高、体重、健康目标。
  用户注册后这些数据全是空的，营养目标只能使用默认值 2000
  kcal。
